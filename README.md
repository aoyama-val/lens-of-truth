# Lens of Truth and 3D Maze

## Dependencies

- macOS 13.5.2
- clang 15.0.0
- glfw 3.3.9
- glm 0.9.9.8

## Setup

```
$ brew install glfw
$ brew install glm
```

## Build

```
$ make
```

## Control

```
W A S D and Mouse: Move
L-Shift: Enable Lens of Truth
```
